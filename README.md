# IBM Data Analyst Capstone Project

Final project for the IBM Data Analyst Professional Certificate. Applied data analytics skills to analyze real-world datasets and present insights via dashboards.

## Project Tasks
- **Data Collection:** Gathered multi-source datasets  
- **Data Wrangling:** Cleaned and prepared data for analysis  
- **Exploratory Analysis:** Summary statistics & visualizations  
- **Statistical Analysis:** Applied methods to uncover trends  
- **Visualization & Dashboards:** Built interactive dashboards in IBM Cognos Analytics  

## Tools & Technologies
- Jupyter Notebook, Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn)  
- SQL & RDBMS  
- IBM Cognos Analytics  

## Key Insights
- Python dominant in current developer use  
- Rust shows highest future interest growth  

## How to Run
1. Clone this repo  
2. Install requirements: `pip install -r requirements.txt`  
3. Open notebooks in `/notebooks`  
4. View final report in `/report`
